# Documentation du Projet BTS - Gestion Hôtelière

Ce dossier contient les explications techniques détaillées du projet. L'objectif de cette documentation est de faciliter la compréhension du code pour la soutenance et les évolutions futures.

## Structure de la Documentation

1. [Architecture Globale](Architecture.md) : Vue d'ensemble des technologies et de l'organisation des fichiers.
2. [Base de Données (Modèles)](Modeles.md) : Explication de la structure des données (Hôtels, Chambres, Réservations).
3. [Logique Backend (Vues et Routes)](Backend.md) : Détails sur le fonctionnement du serveur Django.
4. [Interface Utilisateur (Frontend)](Frontend.md) : Explication des templates HTML et des interactions.
5. [Scripts Utilitaires](Scripts.md) : Rôle des fichiers de lancement et de configuration initiale.
